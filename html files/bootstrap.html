Blockquotes

If you want to add a quote you can use the blockquote HTML tag. Use classes blockquote or blockquote-reverse to make it pretty. 

            <blockquote class="blockquote-reverse">
                <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
                <footer>Quote by <cite title="John Doe">John Doe</cite></footer>
            </blockquote>

Lists

To remove any list styling (padding and bullets) use list-unstyled.

Inline lists with list-inline.

Contextual colors

  • text-primary: blue.
  • text-success: green.
  • text-info: green.
  • text-warning: brown.
  • text-danger: red.
  • text-muted: gray.

Replace ‘text’ with ‘bg’ to edit the background.

Buttons

Every button implemented needs the following format: class=”btn {btn style you want}” 

            Basic buttons
            <button class="btn btn-default">Button</button>
            Anchors:
            <a href=”…” class=”btn btn-default” role=”button”>Link</a>
            Buttons with a input:
            <input type="submit" class="btn btn-default" value="Submit">

Tables

Similar to buttons, use class="table {rest}".

Table-striped strips every second row.

Table-bordered adds a border on all sides of a cell.

Table-hover makes it so that the row you’re hovering changes colors.

Table-condensed adjusts the content of the table when changing the size of the window.

List Groups

List groups are a flexible and powerful component for displaying a series of content.

Add active or disabled to make it appear as so.

Links and Buttons

Use anchor buttons to create actionable list group items with hover, disabled, and active states by adding list-group-item-action.

Cards

A card is a content container. Includes options for headers and footers, a wide variety of content, contextual background colors, and powerful display options. Built with flexbox, they offer easy alignment and mix well with other Bootstrap components.

Cards are made with an outside div with the class card.

  • card-img-top: class that can be applied to an image that will appear at the top of the card
  • card-body: applied to a div and contains everything in the body
  • card-title: applied to a title tag (h1, h2 usw) and
  • card-text: applied to a paragraph or div or whatever and contains the text of the card

Grids - Page Layout

A grid can have 12 columns. Rows are made with a div with the class row. Each column must have a col-*- class.

The first star is replaced by size, eg:

Nothing for phones, sm for tablets, md for desktops, lg for big desktops, and xl for even larger displays.

The second star represents the number of columns to be used from the grid.

Basic Structure of a Bootstrap Grid

To create the layout you want, create a container. Next, create a row (<div class="row">). Then, add the desired number of columns (tags with appropriate col-*- classes). Note that numbers in .col-*-* should always add up to 12 for each row.

Images & Figures

Responsive Images

Images in Bootstrap are made responsive with the class img-fluid.

<img src="picture of your choice" class="img-responsive" alt="Responsive image">

Rounded Corners

The img-rounded class adds rounded corners to an image.

<img src="picture of your choice" class="img-rounded" alt="pic with rounded corners">

Circle

The img-circle class shapes the image to a circle.

<img src="picture of your choice" class="img-circle" alt="pic with rounded corners" width="304" height="236">

Carousel

Creates a slideshow.

Carousels don’t automatically normalize slide dimensions = you may need to use additional utilities or custom styles to appropriately size content. While carousels support previous/next controls and indicators, they’re not explicitly required.

You must add the .active class to one of the slides; otherwise, the carousel will not be visible. Also, be sure to set a unique id on the .carousel for optional controls, especially if you’re using multiple carousels on a single page. Control and indicator elements must have a data-bs-target attribute (or href for links) that matches the id of the .carousel element.

Also, able to add indicators, which say on which slide one is on.

To make a carousel, first make an outside div. This needs the class carousel slide and the data ride carousel. Then make a carousel inner, in which you make divs for different images and captions. Anchors with the class carousel-control-prev or next can be used to control which slide you are on, with a span with the class carousel-control-prev-icon. Indicators are made with a list with the data-slide-to attribute. One needs to be set to active.

Also, able to add a fade with carousel-fade in the first div.

Dropdowns

Surround everything in a div with the class btn-group.

Make a button with the following things: type="button" class="btn btn-danger dropdown-toggle" data-bs-toggle="dropdown"

The data-bs-toggle is what makes the button drop down.

Then make a list with each of the dropdown items, with the class dropdown-item. Use dropdown-divider to divide two dropdown items.

Pagination

Rounded box with multiple boxes of context inside. Declare a list with a pagination and then the list items with page-items. Then make an anchor or button inside the lis.

Forms

Mostly use <input>. Use the class form-control and have a placeholder for placeholder text. One could also have a textarea, with a configurable number of rows. Also use form-label and form-text. One could also use readonly.
